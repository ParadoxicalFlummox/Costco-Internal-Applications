# Costco Internal Applications
A collection of internal applications built to automate day to day tasks

# Development Disclaimer
This repository contains generic architectural patterns and logic skeletons developed as personal projects for the purpose of skill aquisition and portfolio building.
- R&D (Off the clock work): Development is strictly limited to non-proprietary, black-box logic
- Implementation (On the clock work): The connection of these generic skeletons to specific internal datasets, mailing lists, and Costco specific business logic is performed exclusively during recorded work hours on company equipment.
- This repo only contains personal creations and not the code built for the company, the logic should be easy enough to follow and implement during on the clock hours.
